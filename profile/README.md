## Overview

TinkerTool Advanced is a utility for accessing hidden macOS preferences and system settings that Apple does not expose through the standard System Settings interface. It provides a graphical interface for modifying defaults database entries that are normally accessible only through Terminal commands, making system customization accessible without command-line knowledge. The application modifies only user-level settings, never altering system files.

Hidden preferences expose configuration options for Finder, Dock, Safari, Mail, Terminal, and other system applications that Apple has implemented but not included in the standard settings interface. The hidden settings handle customizations that users frequently request — changing screenshot format and location, modifying Dock behavior, adjusting Finder display options, and configuring Safari developer features.

Safety design ensures that only reversible, user-level preferences are modified, never touching system files or protected system settings. Every change can be undone by resetting the preference to its default value. The application does not require administrator privileges for standard preference modifications.

Organized interface groups hidden settings by application and system area, making it practical to find specific customizations without knowing the technical preference key name. Each setting includes a description of what it controls and what the default value is.

For users who want to customize macOS behavior beyond what System Settings allows without learning Terminal commands and defaults database syntax, TinkerTool Advanced is the customization utility that makes hidden settings accessible through a safe, graphical interface.
